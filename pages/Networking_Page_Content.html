Basics of network communication

What is a packet?

We will start with a packet, a packet is a small unit of data transmitted over the network. This basically means that if i send something over the internet then the data of that information gets broken into smaller data so that its more efficient, reliable and flexible.

  1. Efficiency
    • it allows for multiple devices to share a network, simultaneously as the packets can be transmitted independently.
  2. Reliability
    • if a packet is lost or corrupted during transmission, then only that specific packet needs to be resent, this is good because this helps reduce overall data transmission time
  3. Flexibility
    • Packets can take different paths through a network, ensuring that data can reach its destination even if there are network failures or congestion. Packet Structure:

A typical packet consists of two main parts:

Header: Contains information about the packet, such as: Source and destination addresses, Packet type or protocol, Sequence Number, Error detection code.

Data: The actual content of the message.

Domain Name System

What is a DNS?

DNS Stands for Domain Name System and a URL to IP Translation. WHat does this mean? Well url and IP are interconnected concepts that play a crucial role in how we access websites on the internet.

  1. What is DNS?
    • DNS is a hierarchical distributed naming system, you can think of it like the hierarchy in the romans. Translate human-readable domain names into numerical IP addresses.
  2. The connection to DNS:
    • DNS is the primary tool used for URL-to-IP translation. When your computer needs to access a website, it sends a DNS query to resolve the URL into an IP address

Device addresses

IP Address

An IP address (Internet Protocol address) is a unique numerical label assigned to each device connected to a network that uses the Internet Protocol. It serves as a network address, enabling devices to communicate with each other over the interne

What is IPV4??

IPV4 : stands for (Internet Protocol version 4): This is the older version and uses 32 bits to represent an address

What is IPV6??

IPV6 : stands for (Internet Protocol version 6): This is the newer version and uses 128 bits to represent an address. This is good for for the growing number of devices connected to the internet.

What is a MAC Address??

Mac Address : is a unique identifier assigned to a network interface controller (NIC) in a computer or other networking device. It's essentially the physical address of the device on a network

Purpose of MAC Addressing:??

Addressing : MAC addresses are used to identify devices on a local network.

purpose of Mac Address of Data transmission??

Data transmission : When a device sends data to another device on the same network, the MAC address of the destination device is included in the packet

Basic protocols

What is HTTP/S?

stands for Hypertext Transfer Protocol Secure. It's a protocol used to transmit data on the World Wide Web

  1. Security?
    • HTTPS is more secure than HTTP due to the encryption layer.
  2. URL:
    • HTTPS URLs start with "https://" instead of "http://".
  3. Certificate:
    • HTTPS websites require a digital certificate to establish a secure connection with the web browser.
  4. Cost:
    • Implementing HTTPS can involve additional costs for obtaining a digital certificate and configuring the web server.

POP3

is a protocol used to retrieve email messages from a mail server to a local email client. It allows users to download emails from a server to their computer for offline reading and storage.

What is IMAP?

IMAP : stands for (Internet Protocol version 4): This is the older version and uses 32 bits to represent an address

What is SMTP??

SMTP : (Simple Mail Transfer Protocol) is a protocol used to send emails between servers on the internet. It defines the rules for how email messages are formatted and transmitted.
